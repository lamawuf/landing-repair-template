# Шаблон лендинга: Ремонт квартир

Профессиональный лендинг для компаний по ремонту квартир.
Выглядит на 30 000 ₽ — продавай за 5-20к.

## Технологии

- **HTML5** — семантическая вёрстка
- **Tailwind CSS** — через CDN, без сборки
- **Alpine.js** — квиз, модалки, мобильное меню
- **Lucide Icons** — современные иконки
- **Google Fonts** — Inter + Playfair Display

## Структура

```
repair-niche/
├── index.html     ← Весь лендинг в одном файле
├── README.md      ← Этот файл
└── assets/        ← Папка для изображений
```

## Секции

1. **Header** — фиксированный, с телефоном и CTA
2. **Hero** — главный оффер + 3 цифры доверия
3. **Benefits** — 4 преимущества с иконками
4. **Portfolio** — галерея проектов (6 карточек)
5. **Quiz CTA** — промо-блок с призывом
6. **Reviews** — 3 отзыва клиентов
7. **Contacts** — контакты + форма заявки
8. **Footer** — навигация + соцсети
9. **WhatsApp** — плавающая кнопка
10. **Quiz Modal** — 4-шаговый квиз

## Кастомизация

### 1. Телефон и WhatsApp

Найти и заменить:
```
+7 (999) 123-45-67  →  твой номер
79991234567         →  номер без + для WhatsApp
```

### 2. Название компании

```
РемонтПро  →  название клиента
```

### 3. Цвета

В `<script>tailwind.config</script>`:
```javascript
colors: {
    primary: '#1e3a5f',      // Основной (синий)
    'primary-light': '#2d5a8a',
    accent: '#c9a86c',       // Акцент (золотой)
    'accent-light': '#d4ba8a',
}
```

### 4. Город

```
в Москве  →  в [Город клиента]
```

### 5. Фотографии

Заменить плейсхолдеры в секции Portfolio:
```html
<div class="absolute inset-0 flex items-center justify-center">
    <!-- Заменить на: -->
    <img src="assets/project-1.jpg" alt="..." class="w-full h-full object-cover">
</div>
```

### 6. Yandex.Metrika

Раскомментировать блок в `<head>` и заменить `XXXXXXXX` на ID счётчика:
```html
ym(XXXXXXXX, "init", {...});
```

### 7. Отзывы

В секции Reviews заменить:
- Инициалы (АК, МС, ДВ)
- Имена клиентов
- Названия объектов
- Тексты отзывов
