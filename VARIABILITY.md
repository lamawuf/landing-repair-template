# Вариативность шаблона

Опции для кастомизации под разных клиентов.

---

## HERO секция

| Вариант | Описание |
|---------|----------|
| `hero-split` | Текст слева, фото справа (текущий) |
| `hero-centered` | Текст по центру, фото на фоне |
| `hero-video` | Видео на фоне |
| `hero-slider` | Слайдер с несколькими фото |
| `hero-minimal` | Только текст, без фото |

---

## BENEFITS (преимущества)

| Вариант | Описание |
|---------|----------|
| `benefits-4cols` | 4 карточки в ряд (текущий) |
| `benefits-3cols` | 3 карточки в ряд |
| `benefits-2cols` | 2 колонки с иконками слева |
| `benefits-icons-top` | Иконка сверху, текст снизу |
| `benefits-icons-left` | Иконка слева, текст справа |
| `benefits-numbered` | Номера вместо иконок |
